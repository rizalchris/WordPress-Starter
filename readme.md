WordPress Starter

A simple WordPress starter project with:
✅ Custom theme already set up
✅ Advanced Custom Fields (ACF) / Secure Custom Fields plugin installed

This repo is meant to help you start a new WordPress project quickly without repeating the same setup every time.

Requirements
Make sure your local environment has:
PHP 8.0+ (recommended)
MySQL / MariaDB
Web server (Apache / Nginx)
Local server (Laragon, XAMPP, MAMP, LocalWP, etc.)

Getting Started
1. Clone the Repository
git clone <your-repo-url> wordpress-starter
cd wordpress-starter

Place the folder inside your local web server directory if needed (e.g. htdocs, www, or public_html).
2. Create a New Database
Example:
CREATE DATABASE wordpress_starter;
