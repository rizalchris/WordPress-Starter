<h1>WordPress Starter</h1>

<p>
  A simple WordPress starter project with:
</p>

<ul>
  <li>✅ Custom theme already set up</li>
  <li>✅ Advanced Custom Fields (ACF) / Secure Custom Fields plugin installed</li>
</ul>

<p>
  This repository is meant to help you start a new WordPress project quickly
  without repeating the same setup every time.
</p>

<hr>

<h2>Requirements</h2>

<p>Make sure your local environment has:</p>

<ul>
  <li>PHP 8.0+ <em>(recommended)</em></li>
  <li>MySQL / MariaDB</li>
  <li>Web server (Apache / Nginx)</li>
  <li>Local server (Laragon, XAMPP, MAMP, LocalWP, etc.)</li>
</ul>

<hr>

<h2>Getting Started</h2>

<h3>1. Clone the Repository</h3>

<pre><code>git clone &lt;your-repo-url&gt; wordpress-starter
cd wordpress-starter
</code></pre>

<p>
  Place the folder inside your local web server directory if needed
  (e.g. <code>htdocs</code>, <code>www</code>, or <code>public_html</code>).
</p>

<h3>2. Create a New Database</h3>

<p>Example:</p>

<pre><code>CREATE DATABASE wordpress_starter;
</code></pre>
